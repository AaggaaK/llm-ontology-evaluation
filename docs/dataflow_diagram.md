## 2. Explanation of the flow

- **Ontology sources вҶ’ Instances**  
  Ontology classes, axioms, and metadata are transformed into structured benchmark instances.

- **Instances вҶ’ Prompts**  
  Each instance is paired with a taskвҖ‘specific prompt template (E/M/H difficulty).

- **Prompts вҶ’ Model generation**  
  Models generate outputs for each task and domain.

- **Model outputs вҶ’ Metrics**  
  Five metrics evaluate different correctness dimensions:
  - SFS вҖ” semantic fidelity  
  - LCI вҖ” logical consistency  
  - OFS вҖ” ontology faithfulness  
  - PFS вҖ” provenance grounding  
  - SCS вҖ” structural coherence  

- **Metrics вҶ’ Processed scores**  
  Scores are saved per model and per domain.

- **Processed scores вҶ’ Aggregated results**  
  Final tables, plots, and summaries are produced.

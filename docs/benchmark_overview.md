\# OntoLLM-Benchmark — Overview



OntoLLM-Benchmark is a multidimensional evaluation suite for ontology-grounded

generation in large language models. It spans \*\*four ontology domains\*\*:



\- Biomedical  

\- Educational  

\- General  

\- Environmental  



and covers \*\*five ontology-grounded tasks\*\*:



\- Definition generation  

\- Axiom completion  

\- Hierarchical placement  

\- Structural coherence  

\- Provenance explanation  



Each domain provides:



\- Ontology-derived benchmark instances (`instances.json`)  

\- Domain metadata (`metadata.json`)  

\- Prompt templates for all tasks and difficulty levels  

\- Five theoretically grounded evaluation metrics  

\- A complete, reproducible evaluation pipeline  

\- Scripts for running models, aggregating results, and generating plots
